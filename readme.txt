== Description ==
 
Restrict access to read a post or page on your writing with the Facebook Likes Button. You'll get more fans on Facebook.

== Installation ==

1. Upload the 'like-lock' folder to your '/wp-content/plugins/' directory
2. Active the plugin via your Wordpress admin panel
3. Click Like Lock settings under the settings menu in your Wordpress admin area to set your message.

PERFECT!
